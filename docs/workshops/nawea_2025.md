# 2025 NAWEA / WindTech Workshops

Prior to the [2025 NAWEA / WindTech conference](https://naweawindtech2025.org) at
UT Dallas, NREL hosted a [series of workshops](https://naweawindtech2025.org/side-events/)
related to the following software produced by NREL:
- WEIS
- FLORIS
- HOPP / H2Integrate
- FAST.Farm

Each workshop introduced the software and provided practical suggestions for modeling and analysis.


## FAST.Farm Workshop

Jason Jonkman and Emmanuel Branlard hosted a two-part workshop on FAST.Farm covering an overview
